# Project: Automation Testing and Clean Architecture - Forum API v1

## [Menjadi Back-End Developer Expert dengan JavaScript](https://www.dicoding.com/academies/276/)  
**Becoming an Expert Back-End Developer with JavaScript**

Proyek ini merupakan bagian dari pembelajaran back-end menggunakan JavaScript dengan membangun RESTful API bernama **Forum API v1**, yang dikerjakan melalui platform [Dicoding](https://www.dicoding.com).

---

## 🎯 Required Success Criteria

Untuk dinyatakan lulus, proyek harus memenuhi kriteria berikut:

- The API can add a thread  
- The API can add a comment to a thread  
- The API can delete a comment from a thread  
- The API can retrieve thread details  
- Implementation of automation testing  
- Implementation of clean architecture  

---

## ⭐ Optional Success Criteria

Fitur tambahan untuk meningkatkan kualitas proyek:

- The API can add replies to thread comments  
- The API can delete replies from thread comments
